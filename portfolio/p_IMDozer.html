Cat® MineStar™ Command for Dozing in Los Bronces Project

Caterpillar offer the Cat® MineStar™ Command for Dozing, which allow line-of-sight remote operation for bulldozer.

This was implemented in two bulldozers involving Caterpillar software and hardware (server application, MSSQL server & on board hardware), Cisco Mesh network technology, GPS integration and line-of-sight remote operation room implementation for semi-autonomous operation of multiple dozers at once, training and change management.

This project has safety, increase operation hours and asset protection objectives that were accomplished.

After this project finished, the complete bulldozer fleet (Cat D11T model) was moved to this solution.
