BlockGrade Module Project

Do you want to know the grade of the material that the mining Haul Truck is carrying, which the best information available?

This module for Dispatch use the Block Model to estimate all the fields imported (treatment vary according) for the polygon where the loader is digging.

This new version improved the speed (using better search algorithm), can manage rotated model and give you information close to real-time, allowing to save some load with ore and improve quality in the material grade in the crusher.
